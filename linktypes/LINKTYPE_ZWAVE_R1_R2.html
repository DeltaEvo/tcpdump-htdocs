LINKTYPE_ZWAVE_R1_R2

Packet structure

+---------------------+
|       Home ID       |
|      (4 Octets)     |
+---------------------+
|    Source NodeID    |
|      (1 Octets)     |
+---------------------+
|    Frame Control    |
|      (2 Octets)     |
+---------------------+
| Destination NodeID  |
|     (0-1 Octets)    |
+---------------------+
|       Length        |
|      (1 Octet)      |
+---------------------+
|       Payload       |
|    (0-54 Octets)    |
+---------------------+
|        FCS          |
|     (1 Octet)       |
+---------------------+

Description

Z-Wave is defined in ITU-T G.9959. The MAC header format here is used by Type 1 devices with a supported data rate of 9.6 Kbps (R1) and Type 2 devices with a supported data rate of 40 Kbps (R2).

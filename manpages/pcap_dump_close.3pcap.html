NAME

pcap_dump_close - close a savefile being written to  

SYNOPSIS

#include <pcap/pcap.h>

void pcap_dump_close(pcap_dumper_t *p);
 

DESCRIPTION

pcap_dump_close() closes the ``savefile.''

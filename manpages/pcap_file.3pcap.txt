DDEESSCCRRIIPPTTIIOONN
       ppccaapp__ffiillee(())  returns  the standard I/O stream of the ``savefile,'' if a
       ``savefile'' was opened with ppccaapp__ooppeenn__oofffflliinnee(()), or NULL, if a network

       should,  if  possible, use calls that support large files on the return

Which GPG Key for Which Source Tarball?

libpcap starting with release 1.4.0, tcpdump starting with release 4.4.0 and all tcpslice releases are signed using the GPG key RSA E089DEF1D9C15D0D.

Most older libpcap and tcpdump releases are signed using the GPG key DSA 464D3CEB89E917F3.
